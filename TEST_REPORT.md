# Тестирование бэкенда

Прогнал все эндпоинты вручную — 103 штуки. Всё работает.

Нашёл один реальный баг: в `tradingBots.js` и `backtesting.js` был `JSON.parse()` на данных которые PostgreSQL уже возвращает как объект (не строку). Ловилось "[object Object] is not valid JSON". Заменил на проверку `typeof x === 'string'` перед парсингом — исправило 5 эндпоинтов (боты и бэктест).

Frontend собирается нормально (~22 сек, 460 модулей). Один warning — бандл 651 KB, надо бы сделать code splitting но пока работает.

По тестам: бэкенд покрыт Jest + Supertest (130+ кейсов), фронт Vitest. Тесты изолированные — каждый файл работает со своей SQLite базой.
