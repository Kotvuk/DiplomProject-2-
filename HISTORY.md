# Что делал и в каком порядке

Краткий лог, чтобы не забыть что за чем шло.

## Начало
- Поресёрчил конкурентов (TradingView, CoinGecko, Binance Web) — ни у кого нет нормального AI + трейдинг в одном флаконе бесплатно
- Выбрал стек: React + Vite на фронте, Express + PostgreSQL на бэке. Next.js показался избыточным для SPA, Vue рассматривал но у React экосистема побогаче для финансовых штук

## Бэкенд
- Поднял Express, подключил Neon.tech (serverless Postgres) — удобно, не надо ничего хостить
- Сделал JWT авторизацию. Сначала был только access token, потом добавил refresh. Отдельные секреты для каждого
- Прикрутил bcrypt, заодно написал миграцию с SHA-256 хешей (legacy осталось с ранней версии)
- Написал все индикаторы с нуля — EMA, RSI (метод Wilder, не SMA — разница бывает до 10 пунктов), MACD, Bollinger. Долго дебажил RSI на edge cases

## AI
- Интегрировал Groq API. Намучился с rate limits — решил через пул из 15 ключей с round-robin ротацией
- Сначала была одна модель, потом перешёл на 3 (Kimi K2, DeepSeek R1, Qwen 3 32B) под разные задачи
- Написал промпты для мультитаймфреймового анализа — система тянет данные по 6 TF, считает индикаторы, всё скармливает модели
- Самое интересное — self-learning петля: сигнал → трекинг → результат → AI рефлексия → учёт в следующих промптах

## Реалтайм
- Подключил Binance WebSocket Stream для live цен. Сервер держит одно соединение и раздаёт N клиентам
- При подключении клиент сразу получает snapshot текущих цен, не ждёт следующий тик

## Фронт
- Графики на Lightweight Charts. Инструменты рисования (линии, фибоначчи) пришлось делать кастомно — в библиотеке их нет
- Роутинг без react-router — просто state + AnimatePresence из Framer Motion
- Темизация через CSS variables (27 штук, dark/light), i18n самописный на ~400 ключей (RU/EN)

## Трейдинг фичи
- Paper trading с PnL, алерты со звуком через Web Audio API
- Скринер, хитмап, whale panel (стакан + крупные сделки)
- Бэктестинг с несколькими стратегиями, оптимизация параметров
- Торговые боты (paper)

## Инфра
- PWA с service worker (cache-first для статики, network-first для API)
- Настроил Kubernetes конфиги, CI/CD через GitHub Actions
- Prometheus метрики есть, Grafana дашборды — WIP

## Баги которые запомнились
- PostgreSQL возвращает JSON поля как объекты, а код пытался JSON.parse — ловил "[object Object] is not valid JSON". Починил проверкой typeof
- Binance иногда кидает массив вместо объекта — пришлось везде добавить Array.isArray()
- Safari требует user gesture для AudioContext — обернул в try/catch
