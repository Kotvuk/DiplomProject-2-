3. Исправлен краш PDF экспорта

БылоСтало

Причина: PostgreSQL возвращает Date объект, а не строку. Добавлен обработчик doc.on('error') на оба PDF endpoint.

Как запустить

Windows

Дважды кликнуть start.bat
→ Автоматически установит зависимости
→ Запустит backend (:3001) и frontend (:5173)
→ Откроет браузер

Вручную

# Терминал 1 — Backend
cd backend
npm install
node server.js

# Терминал 2 — Frontend
cd frontend
npm install
npm run dev

Переменные окружения

Скопировать .env.example → .env в папку backend/ и заполнить ключи.
